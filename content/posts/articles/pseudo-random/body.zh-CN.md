大多数由软件生成的**随机数**其实都是 **伪随机**：它们是确定性的输出，只是被设计成去逼近某个目标分布。经典方法包括直接采样、逆变换和拒绝采样。实践中还有两类更偏系统设计的变体尤其有用：用于平滑游戏事件的 **Pseudo Random Distribution (PRD)**，以及用于生成大规模、无重复乱序序列的 **基于 RSA 的置换**。

## 1. PRD：面向玩家的概率公平性

在普通的伯努利模型中，每次攻击都以相同的固定概率 `p` 进行判定。统计上这很干净，但在游戏里体感往往很生硬：玩家会牢牢记住连续五次暴击，或者五十次攻击一次暴击都不出的长黑。

PRD 因 **Warcraft III** 而广为人知，后来又被 **Dota** 采用；它会根据最近的触发历史，动态调整每次尝试的概率：

```text
自上次触发以来第 k 次符合条件的尝试的概率 = min(1, C × k)
```

- `k` 会在每次符合条件但未触发的尝试后递增。
- 一旦触发，`k` 重置为 `0`。
- `C` 是经过校准的常数，用来让**长期平均值**与设计目标一致。

旧引擎通常会预先把 `C` 算进查找表，而不是在运行时求解。Warcraft III 使用 5% 的表格步进，这虽然方便，但对较高的标称触发率会产生明显误差，尤其是在大约 `15%` 以上时。另外，并不是所有动作都应该推进计数器：例如，对防御塔的攻击就可能被标记为不符合条件。

```mermaid
flowchart LR
    A[发生一次动作] --> B{符合 PRD 条件吗？}
    B -- 否 --> A
    B -- 是 --> C[增加未触发计数 k]
    C --> D[按 p = min(1, C × k) 进行判定]
    D --> E{触发了吗？}
    E -- 是 --> F[应用效果并重置 k = 0]
    E -- 否 --> G[保留 k 供下一次符合条件的动作使用]
    F --> A
    G --> A
```

它的核心收益是 <u>分布塑形</u>。PRD **不会**提高一个技能的平均强度；它做的是减少极端运气。Wiki 上的测量数据很直观：在标称 `10%` 触发率下，平均触发间隔依然是 `10` 次尝试，但标准差降到约 `5.06`；普通独立 RNG 则是 `9.50`。在 `80%` 时，平均触发间隔为 `1.50` 次尝试，PRD 的标准差是 `0.50`，普通 RNG 则是 `0.87`。离谱连串越少，玩家感知到的公平性就越强。

| 模型 | 每次尝试的规则 | 连串表现 | 最适用场景 |
|---|---|---|---|
| 普通 RNG | 每次固定 `p` | 方差高 | 模拟、统计纯度 |
| PRD | 未触发后概率递增，触发后重置 | 极端情况更少 | 暴击、闪避、攻击特效 |
| RSA 置换 | 确定性双射，不是按次抽样的概率模型 | 无重复 | 唯一乱序 ID 或数值区间 |

## 2. RSA：无重复的伪随机洗牌

另一个问题完全不同：如果你要生成 `100,000,000` 个**互不重复**的伪随机数，该怎么做？反复采样再去重非常浪费。这个问题本质上是一个**置换**问题。

RSA 提供了一种在模域上的可逆映射：

```text
N = p × q
φ(N) = (p - 1)(q - 1)

选择 e，使得 gcd(e, φ(N)) = 1
求 d，使得 e × d ≡ 1 (mod φ(N))

置换：y = x^e mod N
逆变换：x = y^d mod N
```

当 `p` 和 `q` 是不同的素数时，映射 `x -> x^e mod N` 在模 `N` 的剩余类上是一个双射。这意味着，你可以把一个有序序列变成一个确定性的乱序序列，而且**不会重复**。

例如：令 `p = 3`、`q = 5`，则 `N = 15`，`φ(N) = 8`。选择 `e = 3`；这时 `d = 3`，因为 `3 × 3 ≡ 1 (mod 8)`。部分映射如下：

`2 -> 8`, `3 -> 12`, `4 -> 4`, `5 -> 5`, `6 -> 6`, `7 -> 13`, `8 -> 2`, `9 -> 9`

有一个重要修正：这是一种作用在 `0..N-1` 上的置换，**不是**只局限于原始子区间 `2..9` 的洗牌。如果你需要精确的区间，例如 `1..100,000,000`，可以选取一个比该区间略大的 `N`，然后跳过超出范围的输出；或者使用现代的保格式置换，例如 Feistel 网络。

**PRD 和 RSA 解决的是不同的问题。** PRD 让重复事件在人的感知上更公平；RSA 风格的模幂运算则提供一种可逆、无重复的洗牌方式。都属于伪随机这个大类，但系统目标完全不同。
