## 问题概述
给定一个整数 n，计算小于 n 的质数个数。

## 关键思路：厄拉多塞筛法
厄拉多塞筛法是一种经典的求解 n 以内所有质数的高效算法，时间复杂度为 O(n log log n)。

### 算法描述
1. 初始化一个包含 2 至 n-1 的数列。
2. 从未被标记的最小数开始，该数即为质数。
3. 将该质数的所有倍数标记为非质数。
4. 重复直到所有数都被处理。

此方法避免了对每个数逐一判断其质数性，显著提升效率。

### 举例说明
当 n = 13，集合为 {2, 3, 4, ..., 12}：

| 步骤  | 选择的质数 | 删除的倍数                 | 剩余集合             |
|-------|------------|----------------------------|---------------------|
| 1     | 2          | 4, 6, 8, 10, 12             | {3, 5, 7, 9, 11}    |
| 2     | 3          | 6（已删除），9              | {5, 7, 11}          |
| 3     | 5          | 10（已删除）                | {7, 11}             |
| 4     | 7          | 无                         | {11}                |
| 5     | 11         | 无                         | {}                  |

由此得到质数: {2, 3, 5, 7, 11}。

---

## 实现细节与优化说明
- 使用布尔数组 `bool` 来标记被删除的数，在内存占用方面优于使用哈希集合。
- 注意题目要求是统计 **小于 n** 的质数个数。

## 代码实现（C++）
```cpp
class Solution {
public:
    int countPrimes(int n) {
        if (n <= 2) return 0;
        std::vector<bool> isDeleted(n, false);
        int count = 0;
        for (int i = 2; i < n; ++i) {
            if (!isDeleted[i]) {
                ++count;
                for (int multiple = 2 * i; multiple < n; multiple += i) {
                    isDeleted[multiple] = true;
                }
            }
        }
        return count;
    }
};
```

## 备注
- 内层循环从 `2 * i` 开始，避免将质数本身标记为非质数。
- 推荐使用 `std::vector<bool>` 进行内存管理。
- 时间复杂度：O(nloglogn)，空间复杂度：O(n)。
