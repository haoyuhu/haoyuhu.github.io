## 聚类简介
聚类是无监督的学习方法，旨在将数据划分为若干簇，使得簇内相似度高，簇间相似度低。广泛应用于数据挖掘和知识发现。

### 核心概念
- **层次与划分式聚类：** 层次聚类形成一个嵌套的树结构，而划分式聚类将数据分配到固定数量且互斥的簇中。
- **互斥、重叠与模糊聚类：** 点可以被严格分配到单个簇（硬聚类），也可以按隶属度分配到多个簇（模糊聚类）。隶属度之和为1。
- **完全聚类与部分聚类：** 完全聚类要求所有数据点都分配到簇，部分聚类允许存在噪声点不属于任何簇。

### 簇的类型
- **明显分离型：** 簇间任意点距离大于簇内任意点距离。
- **基于原型：** 点距离本簇中心比其他簇中心更近。
- **基于图：** 点作为节点，边权表示相似度，通过图结构划分簇。
- **基于密度：** 簇为密度相连区域，适用于形状不规则且含噪声的数据。

## 常用聚类算法分类对比
| 算法               | 方法类别         | 是否需要预先指定k | 是否能识别噪声 | 是否适合任意形状 | 典型复杂度          | 适用场景                   |
|--------------------|------------------|------------------|----------------|-----------------|---------------------|----------------------------|
| **k-Means++**       | 划分式原型       | 是               | 否             | 通常不适用      | 时间 O(nkt)，空间 O(n+k) | 球形、分离明显的簇           |
| **模糊C均值(FCM)** | 软划分           | 是               | 否             | 通常不适用      | 时间 O(nkt)，空间 O(nk)    | 存在簇间重叠，需保留隶属度   |
| **凝聚层次聚类**   | 层次聚类         | 合并时不需要       | 有限           | 某些情形可用    | 时间 O(n^3)，空间 O(n^2)   | 小规模数据，关注层次结构     |
| **DBSCAN**          | 基于密度         | 否               | 是             | 是              | 时间 O(n^2)，空间 O(n^2)    | 不规则簇，存在噪声           |

## 经典算法介绍

### 1. k-Means++ (硬划分k均值聚类)
k-Means++是k-means的改良版，通过优化初始中心选择减少陷入局部最优的风险。

**算法步骤：**
1. 随机选取第一个聚类中心。
2. 后续中心根据距离现有中心的远近，以距离平方为权重概率选取。
3. 将所有点分配给最近中心。
4. 计算每个簇的新中心（均值）。
5. 重复3-4步骤，直至中心移动小于阈值。

**优点：** 简单、快速、效果可解释，适合球形簇。

**缺点：** 需先决簇数；对噪声敏感；不适用于复杂形状簇。

**核心代码示例：**
```python
import random
import math

def dist_sq(p1, p2):
    return (p1[0]-p2[0])**2+(p1[1]-p2[1])**2

def kmeans_plus_plus_init(points, k):
    centers = [random.choice(points)]
    for _ in range(1,k):
        dists = [min(dist_sq(p,c) for c in centers) for p in points]
        total = sum(dists)
        probs = [d/total for d in dists]
        r = random.random()
        cum_prob = 0
        for p, prob in zip(points, probs):
            cum_prob += prob
            if cum_prob >= r:
                centers.append(p)
                break
    return centers
```

### 2. 模糊C均值(FCM)
FCM允许样本对多个簇具有不同程度的隶属度，隶属度满足在各簇之和为1，由一个指数m控制模糊度。

**主要步骤：**
- 初始化聚类中心(可用k-means++初始方式)。
- 计算点对中心的隶属度：

  \[ u_{i,j} = \left(\sum_{k=1}^c \left(\frac{d(x_j,v_i)}{d(x_j,v_k)}\right)^{\frac{2}{m-1}}\right)^{-1} \]

- 更新聚类中心：

  \[ v_i = \frac{\sum_{j=1}^n u_{i,j}^m x_j}{\sum_{j=1}^n u_{i,j}^m} \]

- 迭代直至聚类中心收敛。

**特点：** 保留软隶属度，计算复杂度较高。仍需指定簇数，无法识别噪声。

### 3. 凝聚层次聚类
每个数据点初始为单独的簇，反复合并距离最小的簇直到满足簇数限制或合并比例阈值。

**优点：** 可观察多层次结构，无需预设k；对噪声有限容忍。

**缺点：** 计算和存储开销大；合并是不可逆操作，可能过度合并。

### 4. DBSCAN
基于密度连接的聚类算法，参数包括邻域半径`Eps`和最小邻居点数`MinPts`。

- 核心点：邻域内点数≥`MinPts`。
- 边界点：邻域点数不足，但邻域内存在核心点。
- 噪声点：既非核心点也非边界点，保持孤立。

**聚类流程：**
- 密度可达的所有核心点组成同一簇。
- 边界点归属邻域内的核心点簇。
- 噪声点不归入任何簇。

**优点：** 适合任意形状簇，自动识别噪声。

**缺点：** 参数较敏感；计算复杂度相对较高。

## 参数影响总结
| 算法        | 关键参数                   | 参数变大导致                   | 参数变小导致                   |
|-------------|----------------------------|-------------------------------|-------------------------------|
| k-Means++   | 簇数k，容忍度               | 聚类更细，迭代更早终止          | 聚类更粗，迭代更多              |
| FCM         | 簇数k，模糊指数m            | 隶属度更模糊                  | 趋近硬划分                    |
| 凝聚层次聚类 | 目标簇数，链接方式          | 保留更多簇结构                 | 合并更激进                    |
| DBSCAN      | Eps，MinPts                | 簇更大，噪声点更少             | 簇更小，噪声点更多             |
