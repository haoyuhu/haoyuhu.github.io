# Haoyu Portfolio Creator

Use this skill for repo-specific content and release operations in `haoyuhu-portfolio`.

## Source of truth

- `content/config/*.yaml`
  Structured site, profile, resume, project, and assistant data
- `content/posts/garden/<slug>/`
  Note content with `meta.yaml`, `body.zh-CN.md`, `body.en.md`
- `content/posts/articles/<slug>/`
  Article content with the same directory contract
- `public/data.json`
  Generated bundle. Do not edit this file by hand.

## Preferred workflow

1. If the task changes content, edit YAML / Markdown sources first.
2. Prefer the Python CLI over ad hoc scripts:
   - `portfolio profile import`
   - `portfolio content note`
   - `portfolio content article`
   - `portfolio build`
   - `portfolio check`
   - `portfolio release`
3. If the task needs a local UI flow, use the `Creator Studio` tab with the local FastAPI server running.
4. Before finishing, rebuild the bundle and run checks/tests relevant to the change.

## Local API

- `POST /api/profile/import`
- `POST /api/content/note`
- `POST /api/content/article`
- `POST /api/build`
- `POST /api/check`
- `GET /api/preview/bundle`
- `POST /chat`
